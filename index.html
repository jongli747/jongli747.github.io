Projects

A Deep Neural Network system to Predict Invasive Ductal Carcinoma
To predict whether a woman hascarcinoma or not we proposed a ensemble deep neural network. Breast histopathology images were used to train several well known architecture such as ResNet, VGG, DenseNet etc. and achieved 94% accuracy and F1 score was 88% .
DOBHASHI: A Neural Machine Translation system from English to Bengali
The project’s aim was to buildan end to end machine translation system for Bengali. We experimented seq to seq neural network architecture incorporate with different attention based mechanism. An average BLEU score of 0.46 was achieved from Bahdanau’sattention model.
A Machine Learning Approach to Predict Crime in the Context of Bangladesh
In this project, we collected data manually from Bangladesh Police. We clean and analyzed the data to observe the crime pattern in Dhaka Division. We used some classical machine learning algorithm to predict future crime in Dhaka. From Support Vector Machine algorithm we achieved the best accuracy which is 95%.
End to End Face Recognition system using Arcface
The primary focus of this project was to build an end to end face recognition system for both desktop and web application. The model was trained using MobileNet and Arcface loss. PyTorch was used as a deep learning framework. I used Flask for web application and tkinter for desktop application.
GlairTrail- A Student Monitoring System
This was a part of my software engineering course. Here a student can read an article and the system can track how much article s/he read and also track his/her eye gaze movement. This is a webapp developed using Django, Sqlite and OpenCV.
Real Time Face Recognition System using Facenet
Here I used a pre-traied model of facenet and developed a face recognition system only used in webcam. For data pipeline I used Tensorflow framework. This is a simple console based system.
